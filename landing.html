Game modes

Three ways to predict.
One platform.

Every competition runs one of three formats. Pick the one that fits your audience, or run all three simultaneously.

🎯
Last Man Standing

Users pick one result per event. One wrong prediction and they're out. Games resolve in kick-off order, building tension through the entire gameweek. The last survivor wins.

HIGH TENSION SEASONAL ELIMINATION
📊
Score Predictor

Users predict the exact scoreline for every match in the round. Exact scores earn maximum points, correct results earn partial credit. Rewards knowledge and rewards accuracy.

HIGH SKILL WEEKLY OR SEASON TIERED POINTS
📈
Over / Under

Users predict whether total goals in a match will be over or under a set line. Accessible to casual fans, quick to complete, and easy to run across multiple leagues in a single round.

Why operators choose Gameweek

Most users visit once. Free-to-play prediction games give them a reason to come back — every single week.

Retention
A weekly reason to return

Prediction deadlines create a natural habit loop. Users return before each round to predict, then return again to check results. 38 gameweeks means 38 touchpoints in a single season.

Acquisition
Zero cost to play, zero barrier to sign up

Free-to-play removes the friction that kills sign-up funnels. Users register because the game is fun — not because they've been incentivised. That's a better quality user from day one.

First-party data

Which teams do your users care about? Which competitions do they follow? Prediction patterns reveal fan identity at a level that page views never could. All data stays in your workspace.

Integration
